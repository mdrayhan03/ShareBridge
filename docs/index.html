What it is

A private space on your local network

ShareBridge turns any Wi-Fi network into a room where devices can talk and move files directly between each other. One device hosts, the rest discover it automatically, and files stream peer-to-peer — so transfers are fast and nothing ever leaves your network.

📡

Automatic discovery

Devices find each other on the Wi-Fi automatically — no IP addresses to type.

📁

Massive file transfers

Send 20GB+ files directly, peer-to-peer, at full LAN speed.

💬

Real-time group chat

A live, shared chat room for everyone connected on the network.

🚫

Fully offline

No internet, no accounts, no cloud. Everything stays on your network.

🔒

Private by design

Files are shared over secure tokens and never uploaded anywhere.

💻

Cross-platform

Windows, macOS, Linux and Android — from a single app.

How it works

Three steps, no setup

1

Start

Open ShareBridge on one device — it becomes the host and starts broadcasting on the Wi-Fi.

2

Connect

Other devices on the same network discover the host automatically and join the room.

3

Share

Chat and attach files of any size — they transfer directly between devices.
